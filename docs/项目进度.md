## 风险和约束

- 用户端不能启动即强制登录；
- Public 接口不能错误加登录守卫；
- UserAuth / AdminAuth 不能直接相信前端 userId；
- 文件本体不能直接存数据库；
- 数据库结构变更必须通过迁移管理；
- 敏感配置必须使用环境变量。
